# Panda Dojo Arcade Local v1

## Destaques

- Versão local estabilizada para compartilhar e testar sem backend.
- Type Arena consolidada como experiência principal de treino.
- Mapa do Dojo funcional com fases, bloqueios, medalhas e progresso local.
- Recomendações inteligentes e missões diárias baseadas em dados locais.
- Arcade com Panda Keys jogável e protótipo Selos do Teclado.
- Documentação de execução local, QA e publicação via GitHub Pages.

## Páginas

- Home
- Type Arena
- Aprenda
- Mapa do Dojo
- Arcade
- Conta

## Recursos locais

- XP
- Nível
- Histórico
- Tema
- Conquistas
- Missões diárias
- Recomendações
- Recordes
- Progresso das fases

## Ajustes de estabilidade

- Proteções para dados antigos ou inválidos no localStorage.
- Fundo animado pausa corretamente quando a aba fica oculta.
- Teclado virtual evita atualização constante em abas ocultas.
- Documentação de QA criada para revisão manual recorrente.

## Limitações

- Sem backend.
- Sem login real.
- Sem ranking global real.
- Conta ainda visual.
- Alguns minigames ainda são protótipos.
- CDNs externos continuam necessários para jQuery, GSAP e ícones.

## Próximos passos

- Expandir textos reais por fase.
- Evoluir Selos do Teclado.
- Criar ranking real em uma futura arquitetura com backend.
- Adicionar testes automatizados de métricas e gamificação.
- Planejar migração técnica futura somente se o projeto crescer além do escopo estático.
