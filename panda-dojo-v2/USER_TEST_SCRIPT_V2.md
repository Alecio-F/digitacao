# Roteiro de Teste com Usuários — PandaDigitações V2

## Objetivo do teste

Validar se a V2 do PandaDigitações está clara, bonita, funcional e interessante para usuários reais.

O foco não é apenas encontrar bugs, mas entender se a experiência faz sentido.

---

## Orientações para quem vai testar

Obrigado por testar o PandaDigitações.

Durante o teste, observe:

- se você entende o que cada página faz;
- se consegue treinar sem dificuldade;
- se algo parece confuso;
- se algo parece bonito ou interessante;
- se algo quebra ou trava;
- se no celular a experiência funciona bem.

> Os dados de progresso ficam salvos apenas no seu navegador. Sinta-se à
> vontade para treinar à vontade — nada é enviado para nenhum servidor.
